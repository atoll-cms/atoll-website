## Public demo instance

The demo is published from `atoll-starter` via GitHub Pages:

- https://atoll-cms.github.io/atoll-starter

## Scope of the demo

The demo focuses on architecture and editorial workflows:

- rendering and theme behavior
- plugin lifecycle (install/activate)
- content structure and routing
- operations basics (updates, cache, backups)
